## Getting Strated

add `.env` file in the root repository, for example:

```sh
DB_USERNAME=username
DB_PASSWORD=pasword
DB_NAME=sewabuku
DB_NAME_TEST=dbtest
SECRET_KEY=secret123
BCRYPT_COST=14
```

download some packages in `go.mod` using command `go get url/to/package`

run the program `go run main.go`
